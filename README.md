# Restaurant Management System (C)

##  Project Overview
The **Restaurant Management System** is a console-based application developed in **C** as part of the SDF Semester 1 final submission.
It allows restaurant owners and customers to manage menus, orders, logins, and billing efficiently using file handling.
This project demonstrates the practical application of core **C programming concepts** in a real-world use case.
This project was developed to strengthen understanding of file handling and real-world data management using C.


## Technologies Used
- C Programming
- File Handling
- Structures
- Pointers
- Console I/O

---

##  Features
- Owner and Customer login system
- Menu display with prices
- Order placement system
- Automatic bill generation with GST
- File-based data storage
- Multiple restaurant menu support

##  How to Run

### 1. Clone the repository

### 2. Navigate to the project directory

### 3. Compile the program
gcc main.c -o restaurant


### 4. Run the program
./restaurant
